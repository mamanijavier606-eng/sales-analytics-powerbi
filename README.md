# Sales Analytics Dashboard

Proyecto de análisis de ventas desarrollado con SQL Server y Power BI.

## Tecnologías utilizadas

* SQL Server
* Power BI
* Data Modeling
* Data Visualization

## Dataset

El dataset contiene información de ventas incluyendo:

* Ventas
* Productos
* Categorías
* Clientes
* Fechas de venta

Columnas principales:

VentaID
SaleDate
Quantity
UnitPrice
TotalVenta
ProductName
Categoria
ClienteNombre

## Dashboard

El dashboard incluye:

* Ventas Totales
* Ventas por Categoría
* Productos más vendidos
* Tendencia de ventas
* Top clientes
